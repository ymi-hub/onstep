# OnStep 프로젝트 새 맥 세팅

## 1. 저장소 클론
```
git clone https://github.com/ymi-hub/onstep.git
cd onstep
```

## 2. Node.js 버전 확인 (v18 이상 필요)
```
node --version
```
v18 미만이면 https://nodejs.org LTS 버전 설치 안내해줘.

## 3. 의존성 설치
```
cd frontend
npm install
```

## 5. 개발 서버 실행 확인
```
npm run dev
```
http://localhost:3000 에서 정상 동작하는지 확인해줘.

## 6. Firebase CLI 설치 (배포용)
```
npm install -g firebase-tools
firebase login
```

# 프로젝트 현재 상태 (참고)

저장소: https://github.com/ymi-hub/onstep.git
배포 주소: https://onstep-lifeos.web.app
Firebase 프로젝트: onstep-lifeos-v2 (백엔드), onstep-lifeos (호스팅)

## 완성된 기능
- TODAY: 오늘 루틴 체크 + 잔량 자동 차감
- BOX: 제품 등록/관리
- SETUP: 루틴 세션 편집 (단계별 phases 구조)
- LOG: 캘린더 기록
- IMPORT: AI 루틴 텍스트 자동 변환 (Gemini 2.5 Flash)

## 앞으로 할 작업
- Onboarding 화면 (신규 사용자 안내)
- OOTD 코디 섹션 (Phase 2)
- 배포: firebase deploy --only hosting --project onstep-lifeos

## 폴더 구조
onstep/
├── frontend/          ← Next.js 앱 (메인 작업 폴더)
│   ├── app/          ← 페이지 (page.tsx, box/, setup/, log/, import/)
│   ├── components/   ← 공통 컴포넌트 (BottomNav.tsx)
│   ├── lib/          ← Firebase 초기화 (firebase.ts)
│   ├── types/        ← TypeScript 타입 정의
│   └── .env.local    ← 환경변수 (git 미포함, 위에서 생성)
├── design/           ← 기존 HTML 디자인 시안 (레퍼런스, 수정 금지)
├── firebase.json     ← Firebase 호스팅 설정
└── CLAUDE.md         ← 프로젝트 전체 가이드
