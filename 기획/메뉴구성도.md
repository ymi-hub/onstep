## 데이터 분류 3계층 (전 화면 공통 원칙)

| 계층 | 정의 | 코드 필드 | UI 용어 |
|------|------|----------|---------|
| **Domain** | 독립 영토 (물리적 분리) | `domain` | 도메인 탭 |
| **Category** | 도메인 내 수납 칸 | `category` / `tipCategory` | 카테고리 |
| **Tag** | 화면 간 연결 고리 (컨텍스트) | `tags[]` | 태그 |

### 화면별 계층 적용 현황

| 화면 | Domain | Category | Tag |
|------|--------|----------|-----|
| TODAY > OOTD | — | category (카테고리 편집) | — |
| LOG > 라이브러리 | domain (탭) | — | tags (필터 칩) |
| LOG > Life TIP | — | tipCategory (카테고리 탭) | tags |
| LOG > 수집 | — | categoryTags (필터 칩) | tags |
| BOX | domain (탭) | subCategory / category | — |
| SETUP | — | — | — |
