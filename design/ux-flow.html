발견된 문제점 & 개선 방향
문제점 01
로그인이 온보딩에 없음
현재 각 탭 우상단 아바타로만 로그인 가능. 신규 유저가 인증 없이 Today에 진입 → 데이터 저장 안 됨을 나중에 발견
문제점 02
첫 진입 후 무엇을 해야 할지 모름
Today 진입 시 빈 화면 + "Setup에서 등록하세요" 텍스트만 존재. 어떤 순서로 설정하는지 안내 없음
문제점 03
Setup 진입 장벽
세션 날짜·루틴·제품 연결 등 설정 항목이 많아 처음 사용자에게 압도적으로 느껴짐
개선 01
온보딩 마지막에 로그인 삽입
Slide 3 → Google 로그인 → 퀵셋업 → Today 순서로 연결. 첫 실행부터 데이터 연동 보장
개선 02
퀵셋업 2스텝 위저드
Step 1: 세션 기간 선택 (10일/2주/1개월). Step 2: 아침/저녁 시간 설정. 최소한의 입력으로 바로 시작
개선 03
Today 시작 가이드 카드
설정 완료 항목에 체크, 미완료 항목에 바로가기 버튼. 모두 완료 시 가이드 카드 자동 숨김
