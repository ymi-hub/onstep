01
Setlog DNA 분석 → OnStep 매핑
Setlog의 5가지 핵심 원칙을 OnStep 맥락으로 번역
⏱️
시간 기반 포착
매 시간 알림 → 2초 클립 촬영. "지금 이 순간"을 남기는 구조.
→ 아침/저녁 루틴 완료 순간 피부 상태 포착
🎬
자동 컴파일
클립이 자동으로 하루 브이로그로 엮임. 편집 불필요.
→ 데이로그 카드 자동 생성
🪞
하이퍼 리얼리즘
필터·편집 없음. 진짜 일상. "꾸밈없는 나"가 콘텐츠.
→ 매일 아침 피부 진행 사진 (날 것)
📖
스토리 포맷
링 → 클릭 → 하루 스토리. 캘린더가 아닌 스토리 아카이브.
→ LOG 스토리 링 + 데이 카드
👯
친구 그룹 공동 기록
최대 12명이 같은 시간에 각자 클립을 올려 하루를 공유.
→ 루틴 완료 공유 카드 (인스타/카카오)
⚡
초저마찰 UX
알림 → 앱 열기 → 2초 촬영 → 닫기. 총 10초 이내.
→ 스킨 스냅 3탭 포착 (30초 이내)
